# Contributing

Argus is a research project exploring AI agent security, specifically prompt injection vulnerabilities in vision-based QA agents.

## Status

This is a personal learning project, not actively seeking contributions. The code is published as supporting evidence for security research and educational content.

## If You Find Issues

Feel free to open an issue if you find:
- Security vulnerabilities (especially new prompt injection vectors)
- Bugs that prevent the tool from running
- Documentation errors

## If You Want to Experiment

Fork the repo and experiment freely. The architecture is intentionally simple to make it easy to understand and modify.

## Code of Conduct

Be respectful. This is a learning project.
